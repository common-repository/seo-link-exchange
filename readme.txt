This plugin is designed to integrate a WordPress site with the SEO link eXchange network. 

== Description ==

This is a simple plugin to integrate a WordPress site with the SEO link eXchange network. It places a link in the footer or via its own widget and send your link into the SEO link eXchange server to be published on other sites. The network is based on free link exchange. There is no pay version - all the network clients use the network for free.

== Installation ==

1. Install the Wordpress plugin.

 Option 1:	Go to your WordPress plugin Panel click "Add New" search for "SEO Link eXchange". Click on the Install link on the far right under Actions. Cick the install button on the plugin page.

 Option 2: Download it here. Go to plugins in your Wordpress admin panel and click "Add New" select upload and browse for the plugin you just downloaded click "Install Now" to install the plugin.

3. Activate the plugin in your Wordpress Admin panel under plugins

4. Once the plugin is activated you should see a widget in the widget section of the Appearance secion of your Admin Panel.
Just drag and drop that in to your side bar and you are ready to go. If you do not have a side bar then the plugin will automaticly include the client code in the footer of the site.

5. Check setup.
